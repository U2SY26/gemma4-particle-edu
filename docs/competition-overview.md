# Gemma 4 Good Hackathon — 대회 개요

## 기본 정보

| 항목 | 내용 |
|------|------|
| **주최** | Google DeepMind (Kaggle) |
| **상금** | $200,000 |
| **기간** | 2026-04-02 ~ 2026-05-18 (UTC 23:59) |
| **URL** | https://kaggle.com/competitions/gemma-4-good-hackathon |
| **참가자** | 2,746 Entrants / 12 Teams / 13 Submissions (초기) |

## 제출물 (모두 필수)

### 1. Kaggle Writeup
- 1,500단어 이내
- Track 선택 필수
- 아키텍처, Gemma 4 활용 방법, 기술적 선택 이유 설명
- "New Writeup" 버튼으로 생성 → "Submit" 버튼으로 제출
- 팀당 1개만, 수정/재제출 가능

### 2. 영상 (가장 중요)
- 3분 이내
- YouTube에 게시, 로그인 없이 시청 가능
- 문제와 솔루션을 스토리텔링으로 보여줘야 함
- Media Gallery에 첨부

### 3. 공개 코드 저장소
- GitHub 또는 Kaggle Notebook
- 잘 문서화, Gemma 4 구현이 명확해야 함
- Writeup "Attachments" > "Project Links"에 링크

### 4. 라이브 데모
- 작동하는 데모 URL 또는 파일
- 로그인/결제 없이 접근 가능
- Writeup "Attachments" > "Project Links" 또는 "Files"에 첨부

### 5. 커버 이미지
- Media Gallery 필수 첨부

## 평가 기준

| 기준 | 배점 | 설명 |
|------|------|------|
| **Impact & Vision** | 40점 | 영상 기반, 실세계 문제를 얼마나 설득력 있게 해결하는가 |
