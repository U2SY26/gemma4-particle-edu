# Gemma 4 Particle Edu

> Gemma 4 + 3D Particle Physics Simulator for Science Education

**Kaggle Gemma 4 Good Hackathon** submission by [U2DIA](https://www.u2dia.com)

## Overview

An AI-powered 3D particle physics simulator that uses Google's Gemma 4 model to make science education interactive and accessible. Students can describe physical phenomena in natural language and watch them come to life through real-time particle simulations.

## Features (Planned)

- **Gemma 4 AI Chat** — Natural language interface for physics exploration
- **Real-time 3D Simulation** — Verlet integration particle physics engine
- **Educational Explanations** — AI-generated explanations of physical phenomena
- **Material Database** — SI-unit properties for real-world materials
- **Multilingual** — Korean / English support
- **Accessible** — Works on low-spec devices with GPU auto-detection

## Tech Stack

| Technology | Purpose |
|------------|---------|
| [Gemma 4](https://ai.google.dev/gemma) | AI model for natural language understanding |
| [Three.js](https://threejs.org/) | WebGL 3D rendering |
| [Vercel](https://vercel.com/) | Deployment |
