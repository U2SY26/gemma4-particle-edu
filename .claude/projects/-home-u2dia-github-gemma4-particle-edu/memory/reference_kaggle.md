- Kaggle CLI: ~/.local/bin/kaggle (v2.0.0)
- Token: ~/.kaggle/kaggle.json
- 대회: kaggle.com/competitions/gemma-4-good-hackathon
- 제출 명령: `~/.local/bin/kaggle competitions submit -c gemma-4-good-hackathon -f submission.zip -m "message"`
